**版本号**: V4.0.0  
**发布日期**: 2026-04-03

**版本类型**: 大版本

## 更新概述

Phase 4 是 iCloush 智慧工厂管理系统的第四代大版本，以"业财一体化 & 机动物流部"为核心主题，完成了系统从零到一的全栈架构搭建。本版本涵盖后端 FastAPI 服务框架、微信小程序前端框架、用户认证与权限体系、管理会计模块、报销审核模块、发票管理模块、欠票管理模块、物流中台模块、员工管理模块、任务管理模块等十大核心模块的初始实现，并提供 Docker 容器化部署方案。

## 新增功能

| 模块 | 功能 | 说明 |
|------|------|------|
| 系统架构 | FastAPI 后端框架 | Python 3.11 + SQLAlchemy 异步 ORM + MySQL/TiDB 数据库 |
| 系统架构 | 微信小程序前端 | 品牌黑橙设计语言，统一 UI 组件库 |
| 用户认证 | JWT 权限体系 | 多角色支持：超管 / 管理员 / 组长 / 普通员工 |
| 管理会计 | 成本分类体系 | COST_CATEGORIES 标准分类、成本直录台、利润表 |
| 报销审核 | 全流程报销 | 员工提交 → 管理员审核 → 自动生成成本流水 |
| 发票管理 | 发票基础功能 | 发票上传、OCR 识别、发票列表与详情 |
| 欠票管理 | 欠票台账 | 欠票登记、催票提醒、销账匹配 |
| 物流中台 | 车辆与调度 | 车辆管理、排线管理、出车调度 |
| 员工管理 | 员工 CRUD | 员工信息管理、积分体系、技能标签 |
| 任务管理 | 任务打卡 | 任务发布、拍照打卡、水印验证 |
| 部署方案 | Docker 容器化 | Dockerfile + docker-compose 一键部署 |

## 涉及模块

本版本为系统初始版本，涉及全部模块的初始实现。

## 部署步骤

```bash
git clone https://github.com/Savoxinity/iCloushManagementSystem.git
cd iCloushManagementSystem/iCloush_Backend_V1
cp .env.example .env  # 配置数据库连接和密钥
docker-compose up -d --build
```
