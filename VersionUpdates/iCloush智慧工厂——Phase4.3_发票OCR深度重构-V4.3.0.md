# iCloush 智慧工厂 — Phase 4.3 发票 OCR 深度重构 V4.3.0 版本说明

**版本号**: V4.3.0  
**发布日期**: 2026-04-07  
**Git 提交**: `7904465`、`d4ec70e`  
**版本类型**: 功能版本

## 更新概述

Phase 4.3 对发票管理系统进行了深度重构，核心升级包括三个方面：一是 OCR 服务深度集成腾讯云 VatInvoiceOCR，实现购销方完整信息、金额明细、人员信息、货物服务明细、附加信息等全字段提取；二是数据模型层面 Invoice 表新增 20+ 字段以支持全量 OCR 数据存储，包括发票类型标签、查重标记和核验结果等；三是前端发票夹改为华通 APP 风格，报销审核改为三按钮模式（发票通过 / 小票通过 / 驳回），积分变动后置到审核结果展示。

## 新增功能

| 模块 | 功能 | 说明 |
|------|------|------|
| OCR 服务 | 全字段提取 | 购销方信息（名称/税号/地址电话/开户银行） |
| OCR 服务 | 金额明细 | 不含税金额/税额/价税合计/中文大写 |
| OCR 服务 | 人员信息 | 收款人/复核人/开票人 |
| OCR 服务 | 货物明细 | 名称/数量/单价/金额/税额/税率 |
| OCR 服务 | 附加信息 | 机器编号/省份/城市/消费类型/公司印章 |
| 数据模型 | Invoice 表扩展 | 新增 invoice_type_code/label、购销方详细信息、人员字段、items_json、查重/核验字段等 20+ 列 |
| 前端 | 发票夹重构 | 改为华通 APP 风格 UI |
| 报销审核 | 三按钮模式 | 发票通过 / 小票通过 / 驳回 |

## 优化改进
